# Archive

Historical planning artifacts live here only when they still explain useful
context. Archived docs are not current instructions.

Before archiving, move durable decisions into the active knowledgebase:

- `AGENTS.md` for agent operating rules.
- Inline READMEs for source-adjacent contracts.
- `docs/adr/` for long-lived decisions.
- `docs/README.md` for navigation.

Prefer deleting completed plans when all useful knowledge has been captured
elsewhere.
