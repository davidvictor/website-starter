# Active plans

Put active implementation plans here only while they are being executed.

Plan lifecycle:

1. Create or update a focused plan when work is too large to execute from chat.
2. Keep the plan current while work is in progress.
3. When work lands, move durable decisions into ADRs, inline READMEs, AGENTS, or
   source comments.
4. Delete the plan when it no longer carries unique useful context, or archive it
   under `docs/archive/<year>/` with a superseded/completed note.

Completed plans should not remain in this folder.
